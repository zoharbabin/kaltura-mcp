# Kaltura MCP Server Documentation

Welcome to the Kaltura MCP Server documentation. This documentation will help you get started with the Kaltura MCP server, understand its architecture, and learn how to use its features.

## Overview

The Kaltura MCP Server enables Large Language Models (LLMs) to interact with the Kaltura Media Management Platform through the Model Context Protocol (MCP). It provides tools and resources for managing media entries, categories, and users in Kaltura.
