## Prerequisites

Before you begin, make sure you have the following:

- Python 3.10 or higher
- Git
- A Kaltura account with API access
- Basic knowledge of Python and the Kaltura API

## Overview

The Kaltura-MCP Server is a Model Context Protocol (MCP) server that provides intelligent access to Kaltura APIs for Large Language Models (LLMs). It enables LLMs to understand, navigate, and leverage Kaltura's video management capabilities through a standardized protocol.

The server is built on top of the fast-mcp framework and uses the kaltura-client-py SDK to communicate with the Kaltura API. It provides a set of tools and resources that can be used by MCP clients to interact with the Kaltura API.
